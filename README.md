## 프로젝트 개요

**ConnectFit**은 회사 내부 구성원 간의 원활한 소통과 협업을 지원하기 위한  
사내 커뮤니티 웹 서비스입니다.

사내 게시판과 쪽지(메신저) 기능을 제공하여  
부서 간 커뮤니케이션을 활성화하는 것을 목표로 합니다.

---

## 프로젝트 목적

- 하이브리드 클라우드 환경에서 웹 서비스 구축 경험
- 서버, 네트워크, 배포 전반에 대한 이해
- GitHub 기반 팀 협업 및 버전 관리 경험
- 생성형 AI(GPT 등)를 활용한 프로젝트 기획 및 개발 경험

---

## 주요 기능

- 회원가입 / 로그인
- 사내 게시판 (글 작성, 수정, 삭제)
- 사용자 간 쪽지 기능
- 관리자 권한 관리 (확장 예정)
